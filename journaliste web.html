Le journalisme web

Informer rapidement à l’ère du numérique

Qu’est-ce que le journalisme web ?

Le journalisme web consiste à produire et diffuser de l’information sur Internet. Il utilise les sites d’actualité, les réseaux sociaux, les vidéos et les formats interactifs pour informer le public en temps réel.

Le journaliste web doit être rapide, précis et capable de vérifier ses sources tout en s’adaptant aux nouveaux outils numériques.

Les missions du journaliste web

Rédiger des articles

Il écrit des articles clairs et structurés pour informer les lecteurs rapidement.

Vérifier l’information

Il lutte contre les fake news en vérifiant les sources et les faits.

Utiliser le numérique

Il utilise les réseaux sociaux, les vidéos et les contenus interactifs.

Pourquoi choisir le journalisme web ?

Le journalisme web permet de toucher un large public, d’informer rapidement et de s’adapter aux nouvelles technologies. C’est un métier moderne, dynamique et essentiel dans notre société connectée.
